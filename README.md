## Tech Stack

- React.js
- Material-UI v5
- React Router v6
- React Hooks (useState, useEffect)
- Local Storage
- JavaScript (ES6+)

### Usage

1. Run the development server with `npm run dev`.
2. Open http://localhost:5173/ in your browser.
